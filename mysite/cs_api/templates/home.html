Welcome to CloudStack API

A web based solution to manage CloudStack API request and events.

Getting started:
  1. sign up or log in,
  2. this will unlock the page to add a CloudStack user which must be your first step,
  3. then you will be able to make API requests as the chosen user on the chosen server.
  4. If you want to add event triggers, first register an event server,
  5. then you will be able to add event triggers.
